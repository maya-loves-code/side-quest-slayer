# Side Quest Slayer

Side Quest Slayer is a local-first progress journal for tracking personal quests with optional proof photos, reflections, and daily reminders.

Your quest data stays on your device. Side Quest Slayer V1 does not use accounts, ads, analytics, cloud sync, or remote photo storage.
