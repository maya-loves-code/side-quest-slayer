# Demo Data Tools

Side Quest Slayer includes developer-only demo tools for screenshots, testing, and marketing assets.

## Location

Open Settings in a development build or Expo dev session. The `Demo Tools` section appears below `App Info`.

The controls are guarded by `__DEV__`, so they do not appear in production release builds.

## Generate Demo Data

Tap `Generate Demo Data`, then confirm.

This replaces local data on the development build with an active `Writer` quest. The quest includes 14 writing progress entries across January through June 2026, with bundled writer-themed placeholder images copied into local quest photo storage.

## Generate Completed Quest

Tap `Generate Completed Quest`, then confirm.

This adds a completed `Poet` quest to the Trophy Room with 7 progress entries and a completed date after the final entry.

## Clear Demo Data

Tap `Clear Demo Data`, then confirm.

This clears local quests, photos, reminders, and settings on the development build and returns the app to the clean new-quest flow.
